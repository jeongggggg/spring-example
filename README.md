## Spring Boot 실습

### Spring Boot로 블로그 만들기
- API와 REST API
- 블로그 개발을 위한 엔티티 구성
- 블로그 글 작성을 위한 API 구현(Create)
- 블로그 글 목록 조회를 위한 API 구현하기(Read)
- 블로그 글 삭제 API 구현하기(Delete)
- 블로그 글 수정 API 구현하기
- H2 Database 에서 MySQL DB로 변경
- 댓글 정보 조회/생성/수정/삭제하는 API 구현
